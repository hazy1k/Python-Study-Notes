## 3.2 列表&下切&切片&遍历

```python
# 下面我们来学习一下列表

num = 100
nums = [11, 22, 33, 44, 55] # 定义一个列表
names = ["老王", "小明", "小红", "小刚"]
# 我们为什么要使用
# 1. 方便存储和操作数据
# 2. 列表可以包含不同类型的数据
# 3. 列表可以动态调整大小
# 4. 列表一个可以存储多个数据

print(names) # 打印列表
print(names[0])
print(names[2])
# print(names[5]) # 下标越界，会报错

# 同样，列表可以切片
print(names[0:2])

# 那么，遍历是什么意思呢？
# 遍历就是依次访问列表中的每一个元素，并对其进行操作。
# 可以通过while、for循环来遍历列表。

# 遍历列表
for name in names:
    print(name)

i = 0
while i < len(names): # 也可以用len()函数获取列表的长度
    print(names[i])
    i += 1

```

## 3.3 字符串&下标&切片

```python
# 下面我们来学习一下python中字符串

# python中字符串的形式有多种
# 1. 双引号
name = "name"
print(name)

# 2. 单引号
name2 = 'name2'
print(name2)

# 3. 三对双引号
name3 = """name3"""
print(name3)

# 4. 三对单引号
name4 = '''name4'''
print(name4)

# 了解一下下标
# 字符串的下标从0开始，可以用索引来访问字符串中的字符
name = "hello world"
print(name[0])  # h
print(name[1])  # e
# 下标为负数时，表示从字符串末尾开始计数，-1表示最后一个字符
print(name[-1])  # d
print(name[-2])  # l

# 下面再讲解字符串切片

# 案例：从键盘中获取一个文件名（带有后缀，如"abc.txt"），如是.png那么打印可以
# 上传到网站，否则提示：不支持此格式

file_name = input("请输入一个文件名（需要带后缀）：")
if file_name[-4:] == ".png":
    # 如果文件名是.png，则可以上传
    print("此文件是图片，可以上传")
else:
    # 否则提示不支持此格式
    print("不支持此格式")

# 通过上面的案例我们知道了
# 字符串切片就是从原本的字符串中提取我们想要的

# demo1
name = 'abcdef'
print(name[0:3])  # abc ,有点像range(x, y ,z)
print(name[:3])  # abc ,如果只有1个冒号且左边没有数字，默认从0开始
print(name[3:5])  # de ,省略了f
print(name[2:])  # cdef ,右侧没有任何数字，默认到字符串末尾
print(name[2:-1])  # cde ,此时不包括最后一个字符
print(name[1:5:2])  # bd ,步长为2，从1开始，到5结束，间隔2个字符
print(name[::1]) # abcdef ,步长为1，默认从0开始到字符串末尾，间隔1个字符as
print(name[::-1]) # fedcba ,步长为-1，默认从字符串末尾开始到0结束，间隔1个字符
print(name[::]) # 如果只有两个冒号，默认步长为1，从0开始到字符串末尾，间隔1个字符
```

## 3.4 列表操作-增删改查排序

```python
# 学习一些列表的操作
# 列表的增删改查

# 1.增
names = ["王老师", "张老师"]
print(names)
# 添加元素 append() 即在原来的列表后面新添加一个元素
# names.append("老大")
# print(names)

# 扩展元素 extend() 即在原来的列表后面添加多个元素
# names2 = ["老大", "老二"]
# names.extend(names2)
# print(names)

# 插入元素 insert() 即在指定位置插入一个元素
names.insert(1, "老三")
print(names)


# 2.删（del,pop,remove）
# del 根据下标删除元素
names = ["aa", "bb", "cc", "dd"]
print(names)
del names[1]
print(names)

# pop 删除最后一个元素，并返回该元素
ret = names.pop()
print("要删除的数据", ret)
print(names)

# remove 根据值删除元素
names.remove("aa")
print(names)


# 3.改-修改元素
names = ["aa", "bb", "cc", "dd"]
print(names)
names[2] = "EE"
print(names)


# 4.查(in，not in, count)
names = ["aa", "bb", "cc", "dd"]
if "aa" in names:
    print("aa在列表中")
else:
    print("aa不在列表中")

if "ee" not in names:
    print("ee不在列表中")
else:
    print("ee在列表中")

# count 统计某个元素出现的次数
names = ["11", "11", "11", "11", "22"]
print(names.count("11"))
print(names.count("22"))


# 列表排序 sort reverse
# 1.sort() 排序，默认升序
nums = [11, 22, 55, 33, 57, 88, 2, 4, 11, 4]
print(nums)
nums.sort() # 从小到大
nums.sort(reverse=True)
print(nums)
nums.reverse() # 逆序
print(nums)
```
